# TechLibrary

مكتبة تقنية عربية - منصة معرفية للمطورين والمهندسين.

## المميزات

- 📚 مقالات تقنية شاملة
- 🔍 بحث فوري باستخدام Fuse.js
- 🎨 تصميم عصري متجاوب
- 🌙 وضع مظلم
- 📱 متوافق مع جميع الأجهزة
- ⚡ أداء عالي
- 🏷️ تصنيفات ووسوم
- 💾 حفظ المقالات المفضلة
- 📖 تاريخ القراءة

## التقنيات المستخدمة

- Vite
- Vanilla JavaScript (ES Modules)
- Tailwind CSS
- Marked.js
- Highlight.js
- Fuse.js
- GSAP
- Swiper.js

## التثبيت والتشغيل

```bash
# تثبيت الاعتماديات
npm install

# تشغيل بيئة التطوير
npm run dev

# بناء المشروع للإنتاج
npm run build

# معاينة الإصدار النهائي
npm run preview
```

## نشر على GitHub Pages

في PowerShell على Windows، استخدم:

```powershell
powershell -ExecutionPolicy Bypass -File .\deploy.ps1
```

إذا أردت نشرًا بدون الدفع إلى GitHub مباشرة، استخدم:

```powershell
powershell -ExecutionPolicy Bypass -File .\deploy.ps1 -SkipPush
```
